# scala-cli-pascal-triangle

## Description
Find binomial coeffient of Nth.

## Tech stack
- bash
- scala 2.11

## Docker stack
- docker-cli
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
